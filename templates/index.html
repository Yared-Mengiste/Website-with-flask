About Me

Hi, I’m Yared Mengiste

I am a fourth-year Software Engineering student at HILCOE School of Computer Science and Technology. I have a strong foundation in programming, with particular strength in Python and core backend development concepts such as APIs, databases, authentication, and application logic.

I have hands-on experience with backend frameworks including Flask (basic to intermediate level), Express.js, and Spring Boot, which allows me to adapt quickly and learn new frameworks efficiently. I enjoy building practical systems and understanding how software works end-to-end.

My skill set also includes C#, Java, C++, SQL (MySQL, Oracle, SQL Server), and web technologies such as HTML and CSS. I aim to grow as a backend-focused developer while continuously improving my problem-solving and system design skills.
